# "PARKADEMIC"
**Parkademic** es un proyecto de aula que consiste en el desarrollo de un software de gestión para el parqueadero universitario. Su objetivo principal es automatizar y optimizar el control de acceso de vehículos, ofreciendo funciones básicas como el registro del vehiculo, el dueño, tiempo de parqueo, el dinero que se debe cobrar, entre otras mas funciones.  

para la realización de este proyecto usaremos como lenguaje de programacion __Python__, ya que es el lenguaje que se esta aprendiendo en el curso. Esperamos que a través de este proyecto adquiramos conocimineto de los fundamentos principales de la programación, que má adelante necesitaremos.

# VISIÓN 
Parkademic busca la solución a un problema cotidiano de la sociedad universitaria de la UdeA, que es la gestión del espacio en los parqueaderos, resolviendo problemas comunes como: 

__Control ineficiente__:Los registros estan en una base de datos que no esta hecha especificamente para el parqueadero, lo que puede saturar facilmente la base de datos por exceso de información. 

__Control de tiempos__:En la universidad no se cuenta con un software que registre los tiempos de parqueo por lo que tampoco hay un modo de cobro. 

__Saturación de espacios__:No se tiene un registro claro de los vehículos que están ingresando, por falta de datos en tiempo real. 
A mediano plazo.

Aspiramos a que este software no solo sirva como proyecto académico, sino como un prototipo funcional que pueda implementarse en pequeñas escalas ej: facultades o sedes universitarias, con beneficios claros. Parkademic no solo organizará parqueaderos, será un modelo de cómo la programación puede transformar problemas cotidianos en soluciones inteligente.
# REQUISITOS 
## Funcionales 
Nuestro software estará diseñado con muchas capacidades que le permitirán dar un correcto uso al parqueadero la ciudad universitara, facilitando así la gestión del espacio e información, sus principales funciones serán las siguentes:

- El registro de los vehículos que ingresan, esto implica el  nombre del usuario con sus apellidos, documento de identificación, número de la placa y la hora de ingreso. 

- Calcular automáticamente el tiempo de estacionamiento y el total a pagar, cumpliendo con la normativa del parqueadero, es decir, cobro mínimo de una hora, tarifa de una hora por 7000 pesos  y los cuartos de hora con un costo de 1500 pesos, ademas se debe tener en cuenta que los minutos exedidos de una hora se redondearan a cuartos de hora, para su respectivo cobro. 
    
-    Generar un reporte diario de vehículos ingresados y dinero recaudado, esto con el fin de tener una base de datos con toda la información actualizada, de la cantidad de vehículos registrados, retirados, sin retirar, pagos totales de vehículos retirados, tiempo promedio de parqueo, lista de usuarios y vehículos con tiempo de parqueo máximo y mínimo. 
    
 ## No funcionales 
Nuestro software le garantizará a sus usuarios ciertas caracteristicas que no generen inconformidades o malos entendidos, esas caracteristicas serán: 
- Una interfaz intuitiva con un menú claro en español, donde el usuario no tenga ningun problema de interpretación

- Una buena seguridad, solo los usuarios autorizados podrán acceder a los registros, tambien que solo los propietarios registrados puedan salir del parqueadero, y la correcta validación de usuario y la contraseña. 

- El tiempo de compilado de la información será inferior a los 4 segundos, para mantener la satisfacción de los usuarios. 
- Nuestro software seré compatible con los sistemas operativos más comunes: macIOS, Windows y Linux.
